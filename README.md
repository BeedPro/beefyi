# beefyi

Source for the beefyi blog, built with [Eleventy](https://www.11ty.dev/).

Live site: [blog.beed.org.uk](https://blog.beed.org.uk/)

## Local development

Use Node 20 locally:

```bash
npm ci
npm start
```

Create a production build locally with:

```bash
npm run build
```
